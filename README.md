# Gaming Platform

A comprehensive multi-platform gaming application featuring 28 classic games.

## Features

- 28 fully playable games including Chess, Checkers, Poker, UNO, Tetris, and more
- PVP ranking system with matchmaking
- Support for 150+ languages
- Responsive design for mobile, tablet, and desktop
- Dark theme with custom color scheme
- PWA support for offline play

## Games Included

### Classic Board Games
- Chess, Checkers, Reversi, Go, Backgammon
- Tic-tac-toe, Battleship

### Card Games
- Poker (Texas Hold'em), Bridge, UNO

### Puzzle Games
- Sudoku, Mastermind, 15 Puzzle, Bulls & Cows
- Tetris, Tower of Hanoi

### Word Games
- Alias, Charades, Anagrams, Balda

### And More!
- Quiz, Math Game, Dots, Blokus, Carcassonne

## Getting Started

### Development
```bash
npm install
npm run dev
```

### Production Build
```bash
npm run build
npm run start
```
